maplev-mode: an Emacs Mode for Maple Code Developers

maplev-mode is an Emacs major mode for developing Maple code. Its features include:

  • font-lock (highlighting) of Maple keywords
  • automatic indentation
  • syntax checking (via Mint)
  • online Maple help
  • online display of Maple procedures
  • imenu support
  • auto-fill support
Currently it is in beta test. It works properly with NTEmacs on a Windows machine and with GNU Emacs on a Linux box. It has been tested with GNU Emacs 21.2. It is supposed to work with Xemacs but the testing done with Xemacs has been minimal. The manual is badly out of date.

The current version supports Maple R9 (and previous releases of Maple). The indentation routine has been modified (kludged) to work with the Maple R8/9 `in' operator, in all likelihood the difference will not be noticed.

Kudos go to Roland Winkler for making substantial contributions to this package.
